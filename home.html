Project Component

Hardware Component

- PC (for hosting system application & storing people data)
- Arduino (for controlling the hardware components)
- RFID Reader (for reading students/professors RFID cards)
- Relay (for turning on/off lights in lecture hall)
- Small Servo Motor (for locking/unlocking lecture hall door).

Software Component

- Arduino: Embedded C++ Program (for transmitting data from RFID Reader to the PC & controlling the Relay and the Servo Motor)
- Database: SQL Server or MySQL Database (for recording the data)
- Application: .NET Desktop App (for inputting data and outputting reports like the Absence Report).
